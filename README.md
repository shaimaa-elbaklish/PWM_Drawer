  A. *PWM_Drawer_Pconfig.h*

2. **Hardware Abstraction Layer (HAL):**

  A. *GLCD_Pconfig.h:*
3. **Micro-controller Abstraction Layer (MCAL)**:

  A. *PWM_Lconfig.c*
